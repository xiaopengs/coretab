### 3. 缺失功能清单

#### 3.1 高优先级（MVP补充）

- [ ] **相对时间显示** - "3分钟前"、"13小时前"
- [ ] **分组命名** - 允许用户给分组起名字
- [ ] **锁定分组** - 防止误删除
- [ ] **分享为网页** - 生成可分享的链接列表

#### 3.2 中优先级（v1.1）

- [ ] **排除列表** - 设置某些网站不被保存
- [ ] **启动时恢复** - 浏览器启动时自动恢复
- [ ] **右键菜单** - 页面右键保存到CoreTab

#### 3.3 低优先级（v1.2+）

- [ ] **内存节省显示** - 显示节省了多少内存
- [ ] **统计信息** - 保存了多少标签等数据
- [ ] **主题设置** - 深色模式等
