核心功能

每一个功能都围绕真实场景构建。

不做花哨但无用的功能堆砌。CoreTab 的每一项特性都来自日常浏览中的真实痛点。

多窗口分组

标签页按所属浏览器窗口自动分组，全宽分割线清晰分隔每个工作空间上下文。当前窗口与历史窗口一目了然。

自动检测

域名智能聚类

同一域名下的所有标签页自动聚合成卡片。一眼就能看到 GitHub、Stack Overflow 或任何网站的所有打开页面。

智能聚合

快捷操作

一键关闭某域名的所有标签、瞬间跳转到任意标签页、恢复最近关闭的页面 —— 全部无需离开管理界面。

键盘优先

即时搜索

通过标题或 URL 在所有窗口中查找任意标签页。输入即显示结果，支持模糊匹配与高亮定位。

实时响应

会话持久化

你的标签页布局在每次会话中自动保存。重新打开浏览器，一切原封不动 —— 每个窗口、每个标签页都在原位等你。

自动保存

隐私优先

所有数据仅保存在你的本地机器上。无需账号、无需云端同步、无需遥测收集。你的浏览数据只属于你自己。

零云依赖
核心能力

不只是展示，
更是管理。

CoreTab 让你对所有打开的标签页拥有完整的控制力。

按窗口智能分组

自动识别浏览器中打开的多个窗口，用醒目的全宽分割线将不同窗口的标签页隔开。每个分割线显示该窗口的标签数和域名统计，让你快速了解各窗口的内容分布。当前工作窗口会被特别标记，方便你区分焦点区域。

按域名聚合标签

同一网站下的所有标签页被自动归入一张域名卡片。卡片顶部显示站点名称和标签总数，下方以 chip 形式列出每个具体页面。你可以在卡片级别执行批量操作：关闭某域名的全部标签、跳转到该域名的任意页面等。

全局搜索与恢复

在搜索框输入关键词即可跨所有窗口和域名查找标签页，结果即时呈现。最近关闭的标签页记录在独立面板中，支持一键恢复单个或批量恢复。书签管理同样集成其中，统一入口管理你的所有网页资源。

完全本地化运行

CoreTab 的全部数据存储在你的本地设备上，不依赖任何云端服务。没有账号注册流程，没有数据上传，没有行为追踪。安装即用，卸载即走，不留痕迹。适合注重隐私的用户和企业内部部署场景。
